## Skill Installation

Skills are installed via ClawHub or manual placement:

```bash
# Via ClawHub
openclaw skills install <skill-name>

# Manual
cp -r <skill-dir> ~/.openclaw/skills/
```

---

## Skill Configuration

Skills are configured via `~/.openclaw/config.yml`:

```yaml
skills:
  - name: github
    enabled: true
  - name: vercel-deploy
    enabled: true
```
